# Quant-Model-finance

Quantitative Finance Models in R


This project implements several core models used in quantitative finance, including:
Factor models (CAPM, Fama-French)

Time series models (ARMA, ARIMA)

Volatility models (GARCH)

Risk metrics (VaR, Expected Shortfall)

Key Features

Statistical analysis of financial time series

Model estimation and diagnostics

Forecasting of returns and volatility

Risk measurement

File
quant_models.R → main implementation
How to Use
Load your dataset in the data/ folder
Update file paths if needed
Run the script in R
